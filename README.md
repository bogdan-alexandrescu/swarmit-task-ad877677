# Modern Landing Page

A clean, modern single-page landing website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Mobile-first approach that works on all devices
- **Modern Styling**: Gradient accents and clean typography
- **Interactive Elements**: Smooth scrolling, hover effects, and animations
- **Optimized Performance**: Lightweight with embedded CSS and JS
- **Static Site Ready**: Easy deployment to any static hosting service

## Sections

1. **Hero Section**: Eye-catching banner with headline, subheadline, and CTA button
2. **Features Section**: Three feature cards highlighting key benefits
3. **Testimonials Section**: Two customer testimonials with glassmorphism design
4. **Footer**: Simple footer with copyright information

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to the displayed localhost URL

## Deployment

This is a static site and can be deployed to:
- Netlify
- Vercel
- GitHub Pages
- AWS S3
- Any static hosting service

Simply upload the `index.html` file to your hosting service.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Gradients, Animations)
- Vanilla JavaScript (ES6+)
- NPM serve for development
